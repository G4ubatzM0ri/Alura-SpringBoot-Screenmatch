##Screenmatch

> Projeto desenvolvido na Alura com Spring Boot utilizando Java 17 e Maven, sem camada web, com foco em execução via console e manipulação de dados utilizando a biblioteca .

##⚙️ Tecnologias utilizadas
Java 17
Spring Boot 3.5.4
Maven
Jackson Databind
JUnit / Spring Boot Test

##▶️ Como executar o projeto
Pré-requisitos
Java 17 instalado
Maven instalado
IDE (IntelliJ, Eclipse, VS Code, etc.)

#Executar com Maven
mvn spring-boot:run

ou

#mvn clean install
java -jar target/screenmatch-0.0.1-SNAPSHOT.jar

##🧪 Testes

Para rodar os testes:

mvn test
